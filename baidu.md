# 百度小游戏
laya现在加上了对百度小游戏的适配，之前都是用百度开发者工具导入微信项目，然后增加配置让其适配，再手动调整某些不一致的地方。

# banner广告
banner广告为固定宽高，我这边看到的宽高比是3:1，以传入宽度为准，对应的高度需要自行计算下。

# 更新管理器
我这边碰到过这样的情况，卡在更新检查时不继续了，后来发现，`UpdateManager`的`onCheckForUpdate`回调只会在有新版本时才会触发，之前微信版本我都是把下一步工作写在这个回调中，这里无法触发，就卡住了。

# 开放数据域
百度的开放数据域比较严格，真的把许多函数都给禁用掉了，包括像`getSystemInfoSync`这样的，而不幸的是，Laya中的适配过程，有一些是依赖它的，会报错。

这个问题比较复杂，我仔细研究了下，专门建了一个项目用于解决此问题和进行说明，[请看这里](https://github.com/tianhe1986/laya-ts-baidu-opendata)
